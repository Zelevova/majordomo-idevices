Настройка устройства

Для работы модуля необходимо настроить службу «Найти iPhone» на устройстве с которым настраивается взаимодействие

<#LANG_AppleIDs#>

Для доступа к устройствам необходимо ввести AppleID и пароли к ним на соответствующей странице.

В случае добавления нового устройства к уже введенному AppleID можно воспользоваться кнопкой для обновления списка устройств привязанных к данному Apple ID.

Информация об устройствах

На вкладке "Устройства" можно увидеть текущую информацию о устройствах, задать "Интервал проверки" местоположения и выполнить следующие действия:

  • - отправить сообщение
  • - воспроизвести звук
  • - обновить информацию об устройстве
Вызов модуля в своих скриптах/методах

Для отправки текстовых сообщений
include_once(DIR_MODULES . 'idevices/idevices.class.php');
$idevices_module = new idevices();
$idevices_module->sendMessage($name, $message, $subject, $sound);Особенности применения: $subject может быть либо пустым и не будет выводиться, либо любым и на устройстве будет заголовок "Оповещение "Найти (iPhone|iPad...)"

Для отправки звуковых оповещений
include_once(DIR_MODULES . 'idevices/idevices.class.php');
$idevices_module = new idevices();
$idevices_module->playSound($name, $subject);Особенности применения: $subject может быть либо пустым и не будет выводиться, либо любым и на устройстве будет заголовок "Оповещение "Найти (iPhone|iPad...)"

Для определения местоположения устройств
include_once(DIR_MODULES . 'idevices/idevices.class.php');
$idevices_module = new idevices();
$idevices_module->locate($name);

Для изменения интервала опроса устройств
include_once(DIR_MODULES . 'idevices/idevices.class.php');
$idevices_module = new idevices();
$idevices_module->getCheckInterval($name); // чтение
$idevices_module->setCheckInterval($name, $interval); // установка
